TfL Real-Time Lakehouse

Real-time London transport data on your laptop. Airflow ingests TfL bus and tube arrivals and stores them in Parquet. dbt and DuckDB handle transformation, Great Expectations validates every batch, and OpenLineage provides full data lineage tracking — a completely observable, production-pattern lakehouse with zero cloud dependency.

Problem
Live TfL API data is high-frequency, inconsistent, and hard to trust without repeatable validation.
Build
Airflow ingestion, Parquet lake storage, DuckDB/dbt modelling, Great Expectations checks, and OpenLineage tracking.
Result
A local-first lakehouse pattern that demonstrates production data engineering without cloud spend.

Deep Stock Insights — Financial Prediction Platform

Full-stack platform combining N-HiTS neural forecasting, LightGBM, XGBoost, and Monte Carlo simulations to predict prices across 53 assets (crypto, equities, commodities). Build
